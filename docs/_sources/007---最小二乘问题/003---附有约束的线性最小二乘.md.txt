# 附有约束的线性最小二乘

$$
\begin{cases}
0.03 x_1 + 0.28 x_2 = 0.85 \\
0.68 x_1 + 0.70 x_2 = 0.17 \\
0.62 x_1 + 0.62 x_2 = 0.07 \\
0.63 x_1 + 0.61 x_2 = 0.84 \\
\end{cases}
$$

$$
s.t.
\begin{cases}
3x_1+2x_2\le1\\
2x_1+3x_2\le2\\
x_1,x_2\le2
\end{cases}
$$

```matlab
C = [0.03    0.28
     0.68    0.70
     0.62    0.62
     0.63    0.61];
d = [0.85
     0.17
     0.07
     0.84];
A = [3,2
     2,3];
b = [1;2];
lb = [-Inf,-Inf];
rb = [2,2];
[x,~,~,exitflag,~,~] = lsqlin(C,d,A,b,[],[],lb,rb);
```
